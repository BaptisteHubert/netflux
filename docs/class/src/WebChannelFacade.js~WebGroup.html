    WebGroup

    This class is an API starting point. It represents a peer to peer network, simply called a group. Each group member can send/receive broadcast as well as personal messages, invite other persons or bots (see WebGroupBotServer).

    Example:

    // Create a WebGroup with full mesh topology, autorejoin feature and
// specified Signaling and ICE servers for WebRTC.

const wg = new WebGroup({
  signalingURL: 'wss://mysignaling.com'
  iceServers: [
    {
      urls: 'stun.l.google.com:19302'
    },
    {
      urls: ['turn:myturn.com?transport=udp', 'turn:myturn?transport=tcp'],
      username: 'user',
      password: 'password'
    }
  ]
})
